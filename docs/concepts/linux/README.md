# Linux Fundamentals

**Document Version:** 1.0
**Sprint:** 3 — Linux Fundamentals

---

# Purpose

This document serves as the **entry point** for Linux knowledge within the DevOps Engineering Journey repository.

Linux is the operating system that powers the majority of modern servers, cloud infrastructure, containers, Kubernetes clusters, CI/CD runners, and DevOps tooling. This module provides a structured overview of Linux concepts and links to detailed reference documents.

---

# Learning Objectives

By completing this sprint, you should be able to:

* Understand the Linux operating system and its architecture.
* Navigate and manage the Linux filesystem.
* Manage users, groups, permissions, and ownership.
* Monitor and control processes and services.
* Understand memory, storage, and networking fundamentals.
* Write basic shell scripts for automation.
* Install and manage software packages.
* Configure secure remote access using SSH.
* Troubleshoot common Linux issues.
